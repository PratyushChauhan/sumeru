# Install on macOS

From a [GitHub Release](https://github.com/PratyushChauhan/sumeru/releases), download the DMG for your CPU:

| Asset | Machine |
| --- | --- |
| `*_aarch64.dmg` | Apple silicon (`uname -m` → `arm64`) |
| `*_x64.dmg` | Intel Mac (`x86_64`) |

Open the DMG and drag **sumeru** into Applications (or run it from the mounted volume).

Linux AppImage/deb/rpm assets will not run on macOS.
